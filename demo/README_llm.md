# 加班数据处理工具

这个工具用于处理加班数据，将一条审批记录中的多人加班信息拆分为每个人的单独记录，并以JSON格式输出。

## 功能特点

- 支持处理单条加班数据
- 支持批量处理多条加班数据（从文件读取）
- 使用大语言模型自动识别并提取加班人员信息
- 输出标准化的JSON格式数据

## 使用方法

### 处理单条数据

```bash
python llm.py --data "杨春森 2025-07-31 18:00 2025-07-31 20:00 2 支援仓库搬仓库，加班人员4人，张海鹏，马建发，殷顺威，杨孔祥"
```

### 处理文件中的多条数据

```bash
python llm.py --file overtime_data.txt
```

### 不带参数运行（使用示例数据）

```bash
python llm.py
```

## 数据格式

### 输入格式

输入数据可以是任意描述加班情况的文本，系统会自动识别其中的人员、时间和加班内容信息。

### 输出格式

输出为JSON格式，包含以下字段：
- name: 加班人员姓名
- date: 加班日期
- time: 加班时长

## 示例数据文件

项目中包含一个示例数据文件 `overtime_data.txt`，其中每行包含一条加班记录。您可以参考这个文件的格式来准备自己的数据。

## 注意事项

- 确保网络能够连接到模型服务器
- 处理大量数据时会有一定延迟
- 模型识别结果可能会有误差，请检查输出结果
